# Changelog

All notable changes to the Hyperswitch PrestaShop module will be documented in this file.

## [1.0.0] - 2024-03-25

### Added
- Initial release of Hyperswitch PrestaShop module
- Payment processing integration with Hyperswitch API
- Admin configuration interface
- Webhook handler for payment status updates
- Support for multiple currencies
- Order status management
- Comprehensive error handling and logging
- Multi-language support
- Security features including webhook signature verification

### Features
- Create payments through Hyperswitch
- Capture authorized payments
- Process refunds
- Handle 3D Secure payments
- Process webhook notifications
- Admin panel configuration
- Front-end payment interface
