# Update Elimination Config

```bash
curl --location "$BASE_URL/rule/update" \
  --header "$AUTH_HEADER" \
  --header "Content-Type: application/json" \
  --data '{
    "merchant_id": "merchant_demo",
    "config": {
      "type": "elimination",
      "data": {
        "bucketSize": 10,
        "eliminationThreshold": 0.15
      }
    }
  }'
```

```json
{ "merchant_id": "merchant_demo", "config": { "type": "elimination" } }
```
