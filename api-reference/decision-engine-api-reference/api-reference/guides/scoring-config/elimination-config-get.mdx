# Get Elimination Config

```bash
curl --location "$BASE_URL/rule/get" \
  --header "$AUTH_HEADER" \
  --header "Content-Type: application/json" \
  --data '{
    "merchant_id": "merchant_demo",
    "algorithm": "elimination"
  }'
```

```json
{ "merchant_id": "merchant_demo", "config": { "type": "elimination" } }
```
