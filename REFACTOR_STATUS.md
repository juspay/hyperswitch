# Refactor Status: Moving Conversion Trait from domain_models to storage_impl

This refactor aims to remove the `diesel_models` dependency from `hyperswitch_domain_models` by moving the `Conversion` trait and all its implementations to `storage_impl`. This enables parallel compilation with `hyperswitch_connectors`, improving overall build times.

**Files needing updates**:

**Decision**: Only v1 implementations were moved. v2 can break and will be fixed separately.
